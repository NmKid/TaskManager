# 📝 Feedback Backlog

このファイルは、テスト中や運用中に気づいた仕様変更案、バグ、アイデアなどを蓄積する場所です。

## 運用ルール
*   ユーザーがチャットで **「バックログへ」** と発言した内容は、AIが自動的にこのファイルに追記します。
*   AIが文脈から「これは修正案・要望として残すべき」と判断した場合も、ユーザーに確認した上で追記することがあります。
*   ある程度溜まったタイミングで、ユーザーが「一括修正」を指示し、AIがまとめて対応します。
